Linear Regression Explained

Linear Regression is a fundamental statistical method used to model the relationship between a dependent variable (Y) and an independent variable (X).

Equation of a Line

The equation for simple linear regression is:

Y = mX + C

  • m: Slope of the line (weight of X).
  • X: Independent variable.
  • C: Intercept (bias), the value of Y when X = 0.

Mathematical Intuition

The goal is to minimize the error between actual and predicted values.

The error is calculated as:

Error = (Y - Y^)

To minimize the error, we use the Sum of Squared Errors (SSE):

SSE = Σ (Yi - Y^i)²

The optimal values of m and C are found using:

m = Σ (Xi - X̄)(Yi - Ȳ) / Σ (Xi - X̄)²

C = Ȳ - mX̄

Why Minimize Squared Error?

Squaring the error avoids negative values canceling out, and since SSE forms a quadratic function (parabola), it guarantees a unique minimum point.

Advantages & Disadvantages

  • Advantages: Easy to interpret, efficient for small datasets.
  • Disadvantages: Assumes linearity, sensitive to outliers.
