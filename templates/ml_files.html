Basic Machine Learning Concepts

1. Training: Learns the relationship between input and output.
2. Testing: Evaluates prediction on unseen data.
3. Model: Maps inputs to outputs.
4. Error: Difference between actual and predicted.
5. Supervised Learning: Uses labeled data.
6. Underfitting: Model too simple.
7. Overfitting: Model too complex.
8. Bias: Missed patterns.
9. Variance: Overreaction to small changes.
10. Regularization: Prevents overfitting.
