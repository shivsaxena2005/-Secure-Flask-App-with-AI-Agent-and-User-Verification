Basic Machine Learning Concepts

1. Training: Finds a function that reduces error, learning the relationship between input and output.
2. Testing: Evaluates how well the model predicts unseen data.
3. Model: A mathematical function mapping inputs to outputs.
4. Error: The difference between actual and predicted values.
5. Supervised Learning: Uses labeled data (input-output pairs).
6. Underfitting: A too-simple model that fails to learn relationships.
7. Overfitting: A too-complex model that memorizes training data.
8. Bias: Inability to capture relationships in data.
9. Variance: Sensitivity to small variations in training data.
10. Regularization: Prevents overfitting by adding constraints.
